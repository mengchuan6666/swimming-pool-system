#### 系统的功能介绍

系统的权限分为前台和后台。

前台的功能：登录、注册、场地查看、列表查看、课程查看、公告查看、在线预约场地

后台的功能：登录、用户管理、场地管理、预约信息管理、 报名信息管理、会员信息管理、数据统计
